 ******** Instalar Modulos
1. npm install express  (framework de NODE codigo con pocas lineas)
2. npm install ejs (motor de plantillas, mejora al html-condicionales etc)
3. npm install morgan (ver por consola lo que los navegadores estan pidiendo al servidor - Logs de peticiones)
4. npm i nodemon -D  (-D dependencia de desarrollo
			Cada vez que realizo un cambio reinicia el servidor)
Crear Carpeta SRC para llevar orden y control de los archivos.

******** Archivo package.json
crear script
	"start" : "node src"
	"dev" : "nodemon src"  (ejecutar el index de la carpeta src al ejecutar
	npm run dev
